Особенности реализации сервера.

- Для обработки запросов и сбора высокоуровневой статистики (редиректы) используется класс HttpRequestHandler.

- Для сбора низкоуровневой статистики (запросы, IP-адреса, количество полученных/отправленных байтов) используются унаследованные классы StatisticInboundHandler StatisticOutboundHandler.

- Аккумулятором статистики является объект класса StatisticCollector.

- Статистика уникальных запросов хранится в хэш-наборе объектов класса ProcessedRequest

- Статистика обработанных соединений хранится в связанном списке объектов класса ProcessedConnection. Методы записи и доступа к коллекциям объектов статистики синхронизированы.

- Форматирование вывода статистики осуществляется в классе StatFormatter.

Существующие найденные проблемы:
- При выбросе исключения в процессе привязки сервера к сетевым соединениям останавливаются процессы сервера, но не головной запускающий процесс.
- При запуске на JVM Java 7 есть конфликт с IPv6, решается запуском сервера с ключем -Djava.net.preferIPv4=true
